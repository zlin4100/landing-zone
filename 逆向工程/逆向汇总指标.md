
# 汇总版：哪些指标还需要逆向


| 模块       | 指标代码                             | 指标名称              | 处理方式   | 原因                                   |
| -------- | -------------------------------- | ----------------- | ------ | ------------------------------------ |
| 增长 + 景气  | GDP_REAL_YOY                     | 中国:GDP:不变价:当季同比   | 可直接喂   | 同比原值本身有较强解释性；但属于季频，仅季末月作为低频确认项               |
| 通胀 + 商品  | BRENT_CRUDE                      | 期货收盘价(连续):ICE布油   | 需月度化   | 原始数据通常为日频，需先统一为月频后再喂模型                        |
| 通胀 + 商品  | XAUUSD                           | 期货收盘价(连续):COMEX黄金 | 需月度化   | 原始数据通常为日频，需先统一为月频后再喂模型                        |
| 流动性 + 信用 | M1_YOY                           | 中国:M1:同比          | 可直接喂   | 同比原值本身已表达方向与强弱，可直接喂                           |
| 流动性 + 信用 | DR007                            | DR007             | 需月度化   | 原始数据为日频资金利率，需先转月均值或月末值                         |
| 流动性 + 信用 | AA_CREDIT_YIELD_3Y               | AA级信用债收益率（3年）     | 需月度化   | 原始数据通常为日频收益率，需先统一为月频                          |
| 流动性 + 信用 | CGB_3Y                           | 中债国债到期收益率:3年      | 需月度化   | 原始数据通常为日频收益率，需先统一为月频                          |
| 需求结构     | CONSUMER_CONFIDENCE              | 中国:消费者信心指数        | 可直接喂   | 指数 level 本身已有较强经济含义，可直接喂                         |
| 需求结构     | INFRA_INVEST_CUM_YOY             | 基础设施投资:累计同比       | 可直接喂   | 累计同比原值已表达方向与强弱，可直接喂                           |
| 需求结构     | MANUFACTURING_INVEST_CUM_YOY     | 制造业投资:累计同比        | 可直接喂   | 累计同比原值已表达方向与强弱，可直接喂                           |
| 需求结构     | REAL_ESTATE_INVEST_CUM_YOY       | 房地产开发投资完成额:累计同比   | 可直接喂   | 累计同比原值已表达方向与强弱，可直接喂                           |
| 需求结构     | PMI_NEW_EXPORT_ORDERS            | 中国:PMI:新出口订单      | 可直接喂   | 50 上下阈值型指标，level 本身有明确经济含义，可直接喂                 |
| 需求结构     | EXPORT_AMOUNT_YOY                | 中国:出口金额:同比       | 可直接喂   | 同比原值本身已表达方向与强弱，可直接喂                           |
| 需求结构     | IMPORT_AMOUNT_YOY                | 中国:进口金额:同比       | 可直接喂   | 同比原值本身已表达方向与强弱，可直接喂                           |
| 需求结构     | SPECIAL_BOND_ISSUE_CUM           | 新增专项债券:累计值        | 需派生后再喂 | 原始累计值随时间单调上升，更适合转成执行进度、当月新增或同比后再喂模型          |
| 需求结构     | LOCAL_SPECIAL_BOND_TARGET_ANNUAL | 政府预期目标:地方专项债      | 需派生后再喂 | 年度目标本身不是月度动态变量，更适合作为专项债执行进度的分母或年度政策锚        |

喂给模型例子：
```csv
indicator_id,ref_month,value,unit,adjustment,source,data_type
CPI_YOY,2024-01,-0.8,%,YoY,NBS,Inflation
CPI_YOY,2024-02,0,%,YoY,NBS,Inflation
CPI_YOY,2024-03,0,%,YoY,NBS,Inflation
```


A. 可直接月频喂原值

这类原值本身就有较强解释性，可直接喂给模型：
- 年度同比类指标：原值已经表达方向与强弱，可直接喂。
- 累计同比类指标：原值已经表达方向与强弱，可直接喂。
- 50 上下阈值类指标：level 本身就有明确经济含义，可直接喂。
- 指数类 level 指标：原值本身已有较强经济解释性，可直接喂。

建议直接喂原值的指标：
- GDP_REAL_YOY（同比原值可直接喂；但它是季频，仅季末月作为低频确认项）
- M1_YOY
- CONSUMER_CONFIDENCE
- PMI_NEW_EXPORT_ORDERS
- EXPORT_AMOUNT_YOY
- IMPORT_AMOUNT_YOY
- INFRA_INVEST_CUM_YOY
- MANUFACTURING_INVEST_CUM_YOY
- REAL_ESTATE_INVEST_CUM_YOY

B. 需月度化

这类指标不建议直接把原始高频数据喂给模型，而应先统一到月频：
- 日频价格类：先转月均值、月末值或月度收益率。
- 日频利率类：先转月均值或月末值。

建议先月度化再喂的指标：
- BRENT_CRUDE
- XAUUSD
- DR007
- AA_CREDIT_YIELD_3Y
- CGB_3Y

C. 需派生后再喂

这类指标原始值不适合直接作为月度宏观判断输入，更适合先做派生特征：
- 累计值类：优先转成当月新增、同比或执行进度。
- 年度目标类：更适合作为分母或年度政策锚，而不是直接当月频动态变量。

建议派生后再喂的指标：
- SPECIAL_BOND_ISSUE_CUM
  - 可派生为：专项债当月新增发行额、专项债累计同比、专项债执行进度
- LOCAL_SPECIAL_BOND_TARGET_ANNUAL
  - 可派生为：SPECIAL_BOND_PROGRESS = SPECIAL_BOND_ISSUE_CUM / LOCAL_SPECIAL_BOND_TARGET_ANNUAL
