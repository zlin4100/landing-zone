## 数学建模核心数据库表（推荐中文名）

### 1. 大类资产年化波动率表
- 对应对象：**波动率向量 `σ`**
- 核心作用：存四大类资产各自的年化波动率，用于描述单资产风险强弱
- 适合承载的数据：
  - 资产类别
  - 观察窗口（1Y / 3Y / 5Y / 10Y / 全样本）
  - 年化波动率
  - 计算口径
  - 生效日期 / 截止日期

### 2. 大类资产年化协方差矩阵表
- 对应对象：**协方差矩阵 `Σ`**
- 核心作用：用于组合风险计算，是资产配置求解的核心输入之一
- 适合承载的数据：
  - 行资产类别
  - 列资产类别
  - 观察窗口
  - 协方差值
  - 年化标识
  - 估计方法 / 计算日期

### 3. 大类资产长期相关性矩阵表
- 对应对象：**相关性矩阵 `ρ`**
- 核心作用：描述四大类资产之间的联动结构，更偏解释、校验和风格识别
- 适合承载的数据：
  - 行资产类别
  - 列资产类别
  - 观察窗口
  - 相关系数
  - 计算口径
  - 截止日期

### 4. 无风险利率期限结构表
- 对应对象：**`r_f` 及利率期限结构**
- 核心作用：提供建模无风险利率基准，以及长短端利率结构信息
- 你当前清单里已经有比较清晰的底层支撑：
  - `CGB_1Y`：建模无风险利率
  - `CGB_3Y`：信用利差分母
  - `CGB_10Y`：长端无风险利率锚
  - 内部合成特征里还定义了 `r_f = CGB_1Y` 和 `TERM_SPREAD_10Y_DR007 = CGB_10Y - DR007`。 
- 所以这张表很适合放：
  - 期限（1Y / 3Y / 10Y）
  - 利率值
  - 基准类型
  - 日期
  - 是否为建模主用利率

### 5. 四大类资产基准画像表
- 对应对象：**Benchmark**
- 核心作用：沉淀四大类资产在不同窗口下的收益、波动、Sharpe 等画像数据
- “画像”这个名字很贴切，因为它不是单一指数映射，而是一组综合特征
- 适合承载的数据：
  - 资产类别
  - 基准指数名称
  - `ann_return_1y / 3y / 5y`
  - `ann_vol_1y / 3y / 5y`
  - `sharpe_ratio_1y / 3y / 5y`
  - 截止日期

### 6. 四大类资产预期收益向量表
- 对应对象：**预期收益向量 `μ`**
- 核心作用：提供优化模型的收益侧输入，和 `Σ`、`r_f` 一起构成求解核心
- 适合承载的数据：
  - 资产类别
  - `mu_1y / mu_3y / mu_5y`
  - `mu_selected`
  - 取值来源
  - 生效日期
  - 备注

## 一句话理解这 6 张表

这 6 张表可以分成三层：

**第一层：风险输入**
- 大类资产年化波动率表
- 大类资产年化协方差矩阵表
- 大类资产长期相关性矩阵表

**第二层：利率基准**
- 无风险利率期限结构表

**第三层：收益画像与优化输入**
- 四大类资产基准画像表
- 四大类资产预期收益向量表
