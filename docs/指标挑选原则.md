# 指标挑选原则（智能投顾 / 资产配置）

在构建智能投顾或资产配置系统时，指标不是越多越好，而是需要遵循清晰的选择原则。
好的指标体系应当 **服务决策、避免冗余、稳定长期，并具备良好的解释能力**。
同时，指标挑选本身也是训练 **宏观理解、信息降维、产品思维和解释能力** 的过程。

---

## 1 指标必须服务一个“决策问题”

每一个指标都必须对应一个明确的投资或业务问题，例如：

| 决策问题   | 指标        |
| ------ | --------- |
| 全球风险偏好 | 标普500、VIX |
| 中国市场趋势 | 沪深300     |
| 通胀压力   | 原油        |
| 避险需求   | 黄金        |

原则：

> **如果一个指标不能改变决策，就不应该采集。**

这一原则训练的是 **产品思维和宏观理解能力**：
通过指标去理解 **经济—市场—资产** 的关系，并让数据直接服务投资决策。

---

## 2 指标不能重复表达同一信息

很多指标之间高度相关，例如：

* SPX
* NASDAQ
* DJI

它们都代表 **美国股票市场风险资产**，信息高度重叠。

正确做法是：

> **每个指标只代表一个核心信息维度。**

例如：

| 信息维度 | 代表指标   |
| ---- | ------ |
| 全球股票 | SPX    |
| 中国股票 | 沪深300  |
| 商品   | 南华商品指数 |
| 利率   | 10年国债  |
| 风险情绪 | VIX    |

这一原则训练的是 **信息降维能力**：
将复杂市场信息压缩为少数关键变量。

---

## 3 指标必须是“市场公认锚点”

优先选择 **全球金融机构普遍使用的指标**，例如：

| 指标         | 角色      |
| ---------- | ------- |
| SPX        | 全球股票市场锚 |
| VIX        | 市场风险情绪  |
| Brent      | 全球原油定价  |
| LME Copper | 工业周期指标  |

优势：

* 市场认可度高
* 数据稳定
* 易于沟通和解释

这一原则强化的是 **宏观理解能力和产品可信度**。

---

## 4 指标必须稳定长期

资产配置和宏观研究需要长期历史数据。

优先选择：

```
历史 ≥ 10年
最好 ≥ 20年
```

长期稳定的指标更适合：

* 宏观周期分析
* 量化回测
* 长期资产配置

这一原则训练的是 **结构化思维和长期视角**。

---

## 5 指标必须可解释

智能投顾不仅需要分析市场，还必须能够 **解释市场**。

好的指标应具备直观含义，例如：

| 指标  | 含义      |
| --- | ------- |
| VIX | 市场恐慌程度  |
| 黄金  | 避险资产    |
| 原油  | 通胀与经济周期 |

避免使用：

* 难以理解的复杂因子
* 不透明指数
* 难以对客户解释的数据

这一原则训练的是 **解释能力和金融沟通能力**。

---

✅ **总结**

好的指标体系应满足五个核心原则：

```
服务决策
避免重复
市场锚点
长期稳定
清晰可解释
```

在实践中，一个成熟的资产配置或智能投顾系统通常只需要 **10–30 个核心指标**，即可有效描述市场并支持投资决策。
