```

  monthly/

    cash_mmf_yld.xlsx                # B5：货基月均收益率
```

## 月度操作 SOP

```
1. 月初等统计局数据发布（通常次月15日前）
2. Choice 导出 B1a → monthly/nbs_macro_core_YYYYMMDD.xlsx + monthly/macro_YYYYMMDD.xlsx
3. Choice 导出 B1b → monthly/nbs_macro_noncore_YYYYMMDD.xlsx
4. Choice 导出 B2  → daily/china_bond.xlsx（更新至月末）+ daily/rates_YYYYMMDD.xlsx
5. Choice 导出 B3  → daily/market_YYYYMMDD.xlsx（更新至月末）
6. FRED 自动拉取 B4 → daily/fred_YYYYMMDD.csv
7. B5 货基收益率随 Choice 更新 → monthly/cash_mmf_yld.xlsx
8. B6 事件日历按实际发生时间追加（不定期）
9. 运行 ETL：
     python -m src.etl.load_macro_monthly --file data/raw/monthly/nbs_macro_core_YYYYMMDD.xlsx
     python -m src.etl.load_macro_monthly --file data/raw/monthly/nbs_macro_noncore_YYYYMMDD.xlsx
     python -m src.etl.load_macro_daily   --file data/raw/daily/china_bond.xlsx
     python -m src.etl.load_macro_daily   --file data/raw/daily/rates_YYYYMMDD.xlsx
     python -m src.etl.load_market_daily  --file data/raw/daily/market_YYYYMMDD.xlsx
10. 构建快照：
     python -m src.aggregation.build_snapshot --month YYYY-MM
```
