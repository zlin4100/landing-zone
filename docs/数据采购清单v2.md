## 结构说明

本版清单沿用《市场分析PRD》的业务分析链路组织：

1. **先判断宏观与政策环境**：市场现在处于什么阶段，有没有系统性风险。
2. **再拆解增长动能来源**：消费、投资、出口谁强谁弱。
3. **再看国际环境传导**：海外利率、美元、油价、黄金、风险偏好如何影响国内。
4. **最后落到四大类资产驱动与建模**：现金类、固收类、股票类、另类资产如何配置与解释。

本版清单对“市场解读”能力的建设路径，采用 **结构化数据 + LLM 推理** 的最小闭环：

- **结构化数据** 负责提供环境、政策、行情与资产结果锚点；
- **宏观热点与主题事件文本** 负责提供市场主线、事件标签与叙事线索；
- **LLM** 负责对宏观、政策、行情、热点之间的关系做归因推理，生成可解释的市场分析结论。

因此，本版弱化“资讯全文主库”思路，不再将财经新闻全文作为 P0 必采项，而是优先保证“宏观量化数据 + 政策事件数据 + 大类资产行情数据 + 宏观热点文本”四类输入闭环。

### 补充说明：以“霍尔木兹海峡长期封锁”为例的解读闭环示例

以下为一个用于说明本清单能力边界的**假设性悲观情景**，用于展示“结构化数据 + LLM 推理”如何支撑市场解读与资产配置解释。

**假设情景**：霍尔木兹海峡长期封锁，布伦特原油持续维持 **120-130 美元/桶**。

在该情景下，系统可优先调用以下已采购/已加工数据，形成“冲击来源 → 国内传导 → 政策约束 → 资产结果”的四层证据链, LLM 可输出如下解读链路：

```text
霍尔木兹海峡长期封锁
→ 原油供给冲击，BRENT_CRUDE 持续高位
→ 输入型通胀压力上升，PPI/CPI 面临抬升
→ 企业成本上升，PMI/工业增加值/GDP 修复承压
→ 政策宽松空间受通胀与汇率约束，稳增长力度受限
→ 股票类承压，债券对冲能力下降
→ 黄金与商品等另类资产分散价值上升
→ 传统股债组合稳定性下降，需提高防御与分散配置权重
```

## 附录 B：删改原则说明

1. **按业务问题分类，不按技术口径分类**：更贴近《市场分析PRD》的分析链路。
2. **避免同义重复**：删除“固定资产投资同比增速”，保留投资三分法。
3. **保留前瞻 + 结果配对**：出口保留 `PMI新出口订单` + `出口累计同比增速`。
4. **信用风险明确化**：新增 `AA级信用债收益率（3年）` 和 `3年期国债收益率`，直接支持信用利差。
5. **四大类资产口径统一**：现金/固收/股票/另类统一按平安银行定义执行。
6. **利率指标按用途分层**：`DR007`、`LPR_1Y`、`RRR`、`CGB_10Y` 放在宏观与政策环境层；量化建模区只尽量保留直接进入协方差矩阵、回测或信用利差计算的利率相关输入。
7. **优先采购原始指标**：涉及专项债执行进度等加工口径时，优先采购”新增专项债累计值 + 政府预期目标:地方专项债”等原始指标，在系统内计算派生指标。
8. **市场解读优先走“结构化数据 + LLM 推理”路线**：弱化财经资讯全文依赖，优先以宏观量化数据、政策事件数据、大类资产行情数据与宏观热点文本构建 P0 解读闭环。
9. **协方差矩阵采用双口径并行**：同时保留方案 A（Total Return）与方案 B（Price Factor），用于比较不同收益定义下的相关性与稳健性。
10. **ETF 退居产品映射层**：`510300`、`518880` 等 ETF 可保留用于产品映射、展示与交易解释，但不再作为协方差矩阵主腿优先口径。
11. **采购表只列原始外部指标**：系统内部聚合、频率转换、收益率换算等加工结果，统一移至“非采购项 / 内部加工序列”说明区，不与采购指标混列，避免产品、数据与研发团队对采购范围产生误解。
