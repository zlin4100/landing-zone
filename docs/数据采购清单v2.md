## 结构说明

本版清单沿用《市场分析PRD》的业务分析链路组织：

1. **先判断宏观与政策环境**：市场现在处于什么阶段，有没有系统性风险。
2. **再拆解增长动能来源**：消费、投资、出口谁强谁弱。
3. **再看国际环境传导**：海外利率、美元、油价、黄金、风险偏好如何影响国内。
4. **最后落到四大类资产驱动与建模**：现金类、固收类、股票类、另类资产如何配置与解释。

本版清单对“市场解读”能力的建设路径，采用 **结构化数据 + LLM 推理** 的最小闭环：

- **结构化数据** 负责提供环境、政策、行情与资产结果锚点；
- **宏观热点与主题事件文本** 负责提供市场主线、事件标签与叙事线索；
- **LLM** 负责对宏观、政策、行情、热点之间的关系做归因推理，生成可解释的市场分析结论。

因此，本版弱化“资讯全文主库”思路，不再将财经新闻全文作为 P0 必采项，而是优先保证“宏观量化数据 + 政策事件数据 + 大类资产行情数据 + 宏观热点文本”四类输入闭环。

### 补充说明：以“霍尔木兹海峡长期封锁”为例的解读闭环示例

以下为一个用于说明本清单能力边界的**假设性悲观情景**，用于展示“结构化数据 + LLM 推理”如何支撑市场解读与资产配置解释。

**假设情景**：霍尔木兹海峡长期封锁，布伦特原油持续维持 **120-130 美元/桶**。

在该情景下，系统可优先调用以下已采购/已加工数据，形成“冲击来源 → 国内传导 → 政策约束 → 资产结果”的四层证据链, LLM 可输出如下解读链路：

```text
霍尔木兹海峡长期封锁
→ 原油供给冲击，BRENT_CRUDE 持续高位
→ 输入型通胀压力上升，PPI/CPI 面临抬升
→ 企业成本上升，PMI/工业增加值/GDP 修复承压
→ 政策宽松空间受通胀与汇率约束，稳增长力度受限
→ 股票类承压，债券对冲能力下降
→ 黄金与商品等另类资产分散价值上升
→ 传统股债组合稳定性下降，需提高防御与分散配置权重
```

### 4.5 协方差矩阵统一收益口径：方案 A / 方案 B

> 目标：为协方差矩阵、MVO、风险平价、BL 等模型提供两套可对比的统一收益口径。两套方案均按平安银行四大类资产定义组织：现金类以流动性管理为核心，固收类以稳健收益为基础，股票类承担长期进攻，另类资产承担分散风险与组合优化。fileciteturn0file0

#### 方案 A：Total Return（投资回报口径）

适用场景：

- 面向真实投资回报的协方差矩阵
- MVO / 风险平价 / BL 的长期配置版本
- 更关注“投资者实际可获得回报”

最严谨版本定义：

- 理想口径：现金管理类总回报指数 / 货币市场工具总回报指数。
- 当前可得且可长期稳定采购的近似口径：`MMF_7D_YIELD_M`（中国货币市场基金月平均七日年化收益率）。
- 使用方式：在系统内转换为月度现金收益，近似处理为 `r_cash_A ≈ MMF_7D_YIELD_M / 12`。
- 解释边界：该指标本质是货币基金收益率的年化表达，不是严格意义上的“现金总回报指数”；但在中国市场缺少统一、长期、可稳定获取的现金类总回报指数背景下，可作为方案 A 现金腿的最稳妥近似口径。

理想资产腿：

- 现金类：`MMF_7D_YIELD_M`（中国货币市场基金月平均七日年化收益率；属于外部原始指标，系统内再转换为月度近似收益）
- 固收类：`CBOND_NEW_COMPOSITE_WEALTH`
- 股票类：`CSI300_TR`
- 另类-黄金：`AU9999`
- 另类-商品：`NHCI`（如后续可得，更优为商品期货总回报指数）

当前相对理想版本的差距：

- 仍可选优化：商品期货总回报指数

#### 方案 B：Price Factor（价格因子口径）

## 附录 B：删改原则说明

1. **按业务问题分类，不按技术口径分类**：更贴近《市场分析PRD》的分析链路。
2. **避免同义重复**：删除“固定资产投资同比增速”，保留投资三分法。
3. **保留前瞻 + 结果配对**：出口保留 `PMI新出口订单` + `出口累计同比增速`。
4. **信用风险明确化**：新增 `AA级信用债收益率（3年）` 和 `3年期国债收益率`，直接支持信用利差。
5. **四大类资产口径统一**：现金/固收/股票/另类统一按平安银行定义执行。
6. **优先采购原始指标**：涉及财政赤字率、专项债执行进度等加工口径时，优先采购“财政赤字（年度）+ GDP（年度，现价）”“新增专项债累计值 + 政府预期目标:地方专项债”等原始指标，在系统内计算派生指标。
7. **市场解读优先走“结构化数据 + LLM 推理”路线**：弱化财经资讯全文依赖，优先以宏观量化数据、政策事件数据、大类资产行情数据与宏观热点文本构建 P0 解读闭环。
8. **协方差矩阵采用双口径并行**：同时保留方案 A（Total Return）与方案 B（Price Factor），用于比较不同收益定义下的相关性与稳健性。
9. **ETF 退居产品映射层**：`510300`、`518880` 等 ETF 可保留用于产品映射、展示与交易解释，但不再作为协方差矩阵主腿优先口径。
10. **采购表只列原始外部指标**：系统内部聚合、频率转换、收益率换算等加工结果，统一移至“非采购项 / 内部加工序列”说明区，不与采购指标混列，避免产品、数据与研发团队对采购范围产生误解。
