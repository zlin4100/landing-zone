## 三、分类调整后的理解要点

### 1. 本次重分类的核心变化

- 不再沿用“宏观周期与政策环境 / 市场动能拆解 / 国际环境 / 大类资产”这种偏业务流程的章节命名。
- 改为更接近投顾研究、宏观配置与资产定价的分类方式，并要求每个原始指标只保留一个归属分类：
  - 经济增长与景气
  - 通胀与价格
  - 消费与居民部门
  - 投资与房地产
  - 外需、贸易与汇率
  - 货币、信用与利率
  - 财政与就业
  - 国际市场与风险偏好
  - 大类资产建模数据
- 其中，`LPR_1Y` 从原“财政、民生与政策工具”调整至“货币、信用与利率”，以体现其在金融条件、信用扩张与贴现率传导中的核心作用。

### 2. 这样分类的好处

- 更接近投资顾问、卖方宏观与资产配置研究的通用口径。
- 更利于后续做指标检索、数据分层、权限管理、看板展示以及“宏观判断 → 资产映射”的解释链路建设。
- 业务解读时仍可按“增长 / 通胀 / 内需 / 投资地产 / 外需 / 流动性信用 / 财政 / 海外风险 / 资产建模”链路输出，但底层数据目录不再混杂业务流程逻辑与指标分类逻辑；同时每个原始指标只保留一个归属分类，避免重复交叉。

### 3. 分类处理原则

处理原则：

- **每个原始指标只归属一个一级分类，不做重复交叉。**
- **若某指标兼具多种解释用途，则按“对资产定价最核心的经济含义”确定其主归属。**
- **政策工具变量优先归入“货币、信用与利率”或“财政与就业”，而不是混合放置。**
- **大类资产建模数据章节只保留上文未出现、但建模额外需要的指标。**

## 五、一句话总结

这版“投资顾问视角最优分类”的重点，不在于逐项机械增删指标，而是把原本按“业务分析流程”组织的采购清单，重排成一套更贴近**投顾研究框架 + 资产定价框架 + 指标库管理框架**的分类方式：

> **上层按投顾研究与资产定价框架分类，底层按原始指标去重，业务侧再按市场分析链路调用。**
